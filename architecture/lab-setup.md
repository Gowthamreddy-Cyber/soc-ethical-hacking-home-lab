# Lab Architecture

## Setup
- Kali Linux (Attacker)
- Windows / Metasploitable (Target)
- Splunk (Monitoring)

## Workflow
1. Launch attack from Kali
2. Target system generates logs
3. Logs are sent to Splunk
4. Analyze logs and detect threats

## Tools Used
- VirtualBox / VMware
- Splunk SIEM
- Kali Linux
